Links & Images in HTML

Links allow navigation between web pages, and images help enhance visual appeal.

1. Creating Links

Use the <a> tag to create hyperlinks.

<a href="https://www.google.com">Visit Google</a>

Attributes:

  • href: Specifies the URL.
  • target="_blank": Opens link in a new tab.
<a href="https://www.google.com" target="_blank">Open Google in New Tab</a>

2. Linking to Other Pages

Use relative paths to link to other pages in your website.

<a href="about.html">Go to About Page</a>

3. Adding Images

Use the <img> tag to display images.

<img src="image.jpg" alt="A beautiful scenery">

Attributes:

  • src: Image source (URL or file path).
  • alt: Alternative text for accessibility.
  • width & height: Define image dimensions.
<img src="image.jpg" alt="Scenery" width="300" height="200">

4. Using Images as Links

Wrap an <img> tag inside an <a> tag.

<a href="https://www.google.com">
    <img src="logo.png" alt="Google Logo">
</a>

Links and images are essential for navigation and visual content in web pages.
