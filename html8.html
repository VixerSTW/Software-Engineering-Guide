Multimedia (Audio & Video) in HTML

HTML allows embedding audio and video files directly into webpages using the <audio> and <video> tags.

1. Adding Audio

Use the <audio> tag to play sound files.

<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    Your browser does not support the audio element.
</audio>

Attributes:

  • controls - Displays playback controls.
  • autoplay - Plays the audio automatically.
  • loop - Repeats the audio.

2. Adding Video

Use the <video> tag to embed videos.

<video controls width="500">
    <source src="video.mp4" type="video/mp4">
    Your browser does not support the video tag.
</video>

Attributes:

  • controls - Displays playback controls.
  • autoplay - Starts playing automatically.
  • muted - Mutes the video by default.
  • loop - Repeats the video.

3. Embedding YouTube Videos

Use an <iframe> to embed videos from YouTube.

<iframe width="560" height="315" src="https://www.youtube.com/embed/VIDEO_ID" frameborder="0" allowfullscreen></iframe>

4. Best Practices for Multimedia

  • Use multiple formats for better browser compatibility.
  • Optimize file size for faster loading.
  • Provide captions for accessibility.

Multimedia elements make web pages more interactive and engaging.
