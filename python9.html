File Handling in Python

Python allows you to read, write, and manipulate files. This is useful for working with data storage and automation.

1. Opening and Reading Files

Use the open() function to open a file. The default mode is read ("r").

# Opening and reading a file
file = open("example.txt", "r")
content = file.read()
print(content)
file.close()

2. Writing to a File

Use "w" mode to overwrite a file or "a" mode to append data.

# Writing to a file
file = open("example.txt", "w")
file.write("Hello, Python!")
file.close()

3. Using with Statement

The with statement automatically closes the file after use.

# Reading a file using with statement
with open("example.txt", "r") as file:
    content = file.read()
    print(content)

4. Working with Multiple Lines

Use readlines() to read a file line by line.

# Reading multiple lines
with open("example.txt", "r") as file:
    lines = file.readlines()
    for line in lines:
        print(line.strip())

5. Checking if a File Exists

Use the os module to check if a file exists before opening it.

import os

if os.path.exists("example.txt"):
    print("File exists!")
else:
    print("File not found.")

File handling is crucial for working with data, logs, and configurations in Python programs.
