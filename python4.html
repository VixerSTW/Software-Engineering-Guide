Conditional Statements

Conditional statements allow us to control the flow of a program by executing certain blocks of code based on conditions.

1. The if Statement

The if statement runs a block of code only if a specified condition is True.

# Example
age = 18

if age >= 18:
    print("You are an adult.")

2. The if-else Statement

The else statement runs a block of code when the if condition is False.

# Example
age = 16

if age >= 18:
    print("You are an adult.")
else:
    print("You are a minor.")

3. The elif Statement

Use elif (short for "else if") to check multiple conditions.

# Example
score = 75

if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
elif score >= 70:
    print("Grade: C")
else:
    print("Grade: F")

4. Nested Conditionals

We can nest conditions inside one another.

# Example
age = 20
has_id = True

if age >= 18:
    if has_id:
        print("You can enter.")
    else:
        print("You need an ID.")
else:
    print("You are too young.")

5. The pass Statement

If you don't want to execute anything inside an if block, use pass as a placeholder.

# Example
x = 10

if x > 5:
    pass  # This does nothing

Conditional statements are crucial for decision-making in Python programs.
