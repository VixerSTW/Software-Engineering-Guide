HTML Headings, Paragraphs & Text Formatting

In HTML, text content is structured using headings, paragraphs, and formatting elements.

1. Headings

Headings range from <h1> (largest) to <h6> (smallest).

<h1>This is Heading 1</h1>
<h2>This is Heading 2</h2>
<h3>This is Heading 3</h3>
<h4>This is Heading 4</h4>
<h5>This is Heading 5</h5>
<h6>This is Heading 6</h6>

2. Paragraphs

Use <p> to create paragraphs.

<p>This is a paragraph of text.</p>
<p>Another paragraph with more content.</p>

3. Line Breaks & Horizontal Rules

Use <br> for line breaks and <hr> for horizontal dividers.

This is line one.<br>
This is line two.

<hr>

This is a new section.

4. Text Formatting

HTML provides various tags for styling text.

<b>Bold Text</b>
<i>Italic Text</i>
<u>Underlined Text</u>
<mark>Highlighted Text</mark>
<small>Small Text</small>
<del>Deleted Text</del>
<ins>Inserted Text</ins>
<sub>Subscript Text</sub>
<sup>Superscript Text</sup>

Text formatting is useful for emphasizing important content in web pages.
