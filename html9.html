Meta Tags & SEO Basics

Meta tags provide information about a web page and improve Search Engine Optimization (SEO).

1. What are Meta Tags?

Meta tags are placed inside the <head> section of an HTML document and help browsers and search engines understand the content.

<head>
    <meta charset="UTF-8">
    <meta name="description" content="Learn HTML, CSS, and web development.">
    <meta name="keywords" content="HTML, CSS, Web Development">
    <meta name="author" content="Your Name">
</head>

2. Important Meta Tags

  • <meta charset="UTF-8"> - Defines character encoding.
  • <meta name="description"> - Gives a short description of the page.
  • <meta name="keywords"> - Provides relevant keywords for search engines.
  • <meta name="author"> - Specifies the author of the page.
  • <meta name="viewport"> - Controls how a page looks on different screen sizes.

3. Viewport Meta Tag (Responsive Design)

The viewport meta tag makes your site mobile-friendly.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

4. Open Graph (For Social Media)

Open Graph meta tags help control how links appear when shared on social media.

<meta property="og:title" content="Learn HTML">
<meta property="og:description" content="A beginner-friendly guide to HTML">
<meta property="og:image" content="image.jpg">
<meta property="og:url" content="https://example.com">

5. Favicon

A favicon is a small icon displayed on browser tabs.

<link rel="icon" href="favicon.ico" type="image/x-icon">

6. Robots Meta Tag

This tells search engines whether to index the page.

<meta name="robots" content="index, follow">

Meta tags improve your site's SEO and how it appears on search engines.
