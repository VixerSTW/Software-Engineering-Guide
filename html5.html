Forms & Input Fields in HTML

Forms allow users to input and submit data on a webpage.

1. Creating a Basic Form

The <form> tag wraps input fields and buttons.

<form>
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    <br>
    <input type="submit" value="Submit">
</form>

2. Input Field Types

HTML provides various input types.

<form>
    <label>Text:</label> <input type="text"><br>
    <label>Password:</label> <input type="password"><br>
    <label>Email:</label> <input type="email"><br>
    <label>Number:</label> <input type="number"><br>
    <label>Checkbox:</label> <input type="checkbox"><br>
    <label>Radio Button:</label> <input type="radio"><br>
    <label>Date:</label> <input type="date"><br>
    <input type="submit" value="Submit">
</form>

3. Dropdowns & Textareas

Use <select> for dropdowns and <textarea> for multi-line text input.

<form>
    <label>Select an option:</label>
    <select>
        <option>Option 1</option>
        <option>Option 2</option>
    </select>
    <br>
    <label>Message:</label>
    <textarea rows="4" cols="30"></textarea>
    <br>
    <input type="submit" value="Send">
</form>

4. Form Attributes

  • action - Specifies where to send form data.
  • method - Defines how to send data (GET or POST).
  • required - Ensures a field must be filled.

Forms are essential for collecting user input on websites.
