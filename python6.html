Functions in Python

Functions allow you to write reusable code, making your programs more efficient and organized.

1. Defining a Function

A function is defined using the def keyword.

# Example: Defining and calling a function
def greet():
    print("Hello, welcome to Python!")

greet()  # Output: Hello, welcome to Python!

2. Function Parameters

Functions can take parameters to accept input values.

# Example: Function with parameters
def greet(name):
    print(f"Hello, {name}!")

greet("Vien")  # Output: Hello, Vien!

3. Returning Values

Functions can return values using the return keyword.

# Example: Function returning a value
def add(a, b):
    return a + b

result = add(5, 3)
print(result)  # Output: 8

4. Default Parameters

You can set default values for parameters.

# Example: Function with default parameter
def greet(name="Guest"):
    print(f"Hello, {name}!")

greet()         # Output: Hello, Guest!
greet("Vien")   # Output: Hello, Vien!

5. Lambda Functions

Lambda functions are small, anonymous functions that can take any number of arguments but have only one expression.

# Example: Lambda function
multiply = lambda x, y: x * y

print(multiply(3, 4))  # Output: 12

Functions help make code reusable and maintainable, making them an essential part of Python programming.
