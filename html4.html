Lists & Tables in HTML

Lists organize content, and tables display structured data.

1. Unordered Lists

Use the <ul> tag for bullet-point lists.

<ul>
    <li>Apple</li>
    <li>Banana</li>
    <li>Cherry</li>
</ul>

2. Ordered Lists

Use the <ol> tag for numbered lists.

<ol>
    <li>Step 1</li>
    <li>Step 2</li>
    <li>Step 3</li>
</ol>

3. Nested Lists

Lists can be nested inside each other.

<ul>
    <li>Fruits
        <ul>
            <li>Apple</li>
            <li>Banana</li>
        </ul>
    </li>
</ul>

4. Creating Tables

Use the <table> tag to create tables.

<table border="1">
    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>Vien</td>
        <td>16</td>
    </tr>
</table>

5. Table Elements

  • <tr> - Table row
  • <th> - Table header
  • <td> - Table data
  • border - Adds a border to the table

Lists and tables help organize content for better readability.
