Lists & Tuples in Python

Lists and tuples allow you to store multiple values in a single variable. They are used to organize and manipulate data efficiently.

1. Lists

A list is a collection that is ordered and changeable (mutable). Lists allow duplicate values.

# Creating a list
fruits = ["Apple", "Banana", "Cherry"]
print(fruits)  # Output: ['Apple', 'Banana', 'Cherry']

# Accessing list elements
print(fruits[0])  # Output: Apple

# Modifying list elements
fruits[1] = "Mango"
print(fruits)  # Output: ['Apple', 'Mango', 'Cherry']

2. List Methods

Python provides various methods to manipulate lists.

# Adding items
fruits.append("Orange")  # Adds 'Orange' to the end
print(fruits)  # Output: ['Apple', 'Mango', 'Cherry', 'Orange']

# Removing items
fruits.remove("Cherry")  
print(fruits)  # Output: ['Apple', 'Mango', 'Orange']

# Sorting a list
fruits.sort()
print(fruits)  # Output: ['Apple', 'Mango', 'Orange']

3. Tuples

A tuple is similar to a list, but it is immutable (cannot be changed after creation).

# Creating a tuple
colors = ("Red", "Green", "Blue")
print(colors)  # Output: ('Red', 'Green', 'Blue')

# Accessing tuple elements
print(colors[1])  # Output: Green

# Tuples are immutable (cannot change values)
# colors[0] = "Yellow"  # This will cause an error

4. Tuple vs. List

Key differences between lists and tuples:

  • Lists: Mutable (can be modified), slower, use more memory.
  • Tuples: Immutable (cannot be changed), faster, use less memory.

5. Looping Through Lists & Tuples

You can use loops to iterate over lists and tuples.

# Looping through a list
for fruit in fruits:
    print(fruit)

# Looping through a tuple
for color in colors:
    print(color)

Lists and tuples are essential data structures in Python, helping you manage and manipulate data efficiently.
