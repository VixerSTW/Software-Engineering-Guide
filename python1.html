Introduction to Python

Welcome to your first Python lesson! Python is a powerful and beginner-friendly programming language used in various fields, such as web development, data science, artificial intelligence, and automation.

Why Learn Python?

  • Easy to Read & Write: Python has a simple syntax similar to English.
  • Versatile: Used in web development, AI, data science, and more.
  • Large Community: Millions of developers use Python, so help is always available.
  • High Demand: Python developers are in demand across various industries.

Writing Your First Python Program

Let's start with a simple program that prints "Hello, World!" to the screen.

print("Hello, World!")

Explanation:

  • print() is a function that displays text on the screen.
  • The text inside the parentheses is enclosed in double quotes " ", which makes it a string.
  • When you run the program, it will output: Hello, World!

Run this code in a Python environment and see the output yourself.
