Variables & Data Types

In Python, variables are used to store data. You can think of them as labeled boxes that hold different types of information.

Declaring Variables

To create a variable, you simply assign a value to a name using the equals sign =.

# Assigning values to variables
name = "Vien"
age = 16
height = 1.68
is_student = True

print(name)
print(age)
print(height)
print(is_student)

Understanding Data Types

Python supports different types of data:

  • String (str): Text enclosed in quotes ("Hello", 'World').
  • Integer (int): Whole numbers (10, -3).
  • Float (float): Decimal numbers (3.14, -2.5).
  • Boolean (bool): True or False values.

Understanding variables and data types is essential for writing efficient Python programs.
