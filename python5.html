Loops in Python

Loops are used to execute a block of code multiple times until a condition is met.

1. The for Loop

The for loop is used to iterate over a sequence (like a list, tuple, or string).

# Example: Loop through a list
fruits = ["Apple", "Banana", "Cherry"]

for fruit in fruits:
    print(fruit)

Using range() in a for loop:

# Example: Loop with a range
for i in range(5):
    print(i)  # Outputs numbers from 0 to 4

2. The while Loop

The while loop runs as long as a specified condition is True.

# Example: Count from 1 to 5
count = 1

while count <= 5:
    print(count)
    count += 1

3. The break Statement

The break statement is used to exit a loop prematurely.

# Example: Stop loop when number is 3
for num in range(1, 6):
    if num == 3:
        break
    print(num)

4. The continue Statement

The continue statement skips the current iteration and moves to the next.

# Example: Skip number 3
for num in range(1, 6):
    if num == 3:
        continue
    print(num)

5. The else Clause in Loops

The else block runs if the loop completes without hitting a break.

# Example
for num in range(1, 4):
    print(num)
else:
    print("Loop completed!")

Loops are essential for automating repetitive tasks in Python.
