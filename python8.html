Dictionaries & Sets in Python

Dictionaries and sets are powerful data structures in Python that allow efficient data storage and retrieval.

1. Dictionaries

A dictionary is a collection of key-value pairs. Each key is unique and maps to a specific value.

# Creating a dictionary
student = {
    "name": "Vien",
    "age": 16,
    "grade": "A"
}

# Accessing values
print(student["name"])  # Output: Vien

2. Dictionary Methods

Python provides several useful methods for working with dictionaries.

# Adding a new key-value pair
student["city"] = "Jakarta"

# Updating values
student["grade"] = "A+"

# Removing a key-value pair
del student["age"]

# Looping through a dictionary
for key, value in student.items():
    print(f"{key}: {value}")

3. Sets

A set is an unordered collection of unique items.

# Creating a set
numbers = {1, 2, 3, 4, 5}

# Adding elements
numbers.add(6)

# Removing elements
numbers.remove(2)

# Checking membership
print(3 in numbers)  # Output: True

4. Set Operations

Sets support mathematical operations like union, intersection, and difference.

# Example of set operations
A = {1, 2, 3, 4}
B = {3, 4, 5, 6}

print(A | B)  # Union: {1, 2, 3, 4, 5, 6}
print(A & B)  # Intersection: {3, 4}
print(A - B)  # Difference: {1, 2}

Dictionaries and sets are essential for storing and managing structured data efficiently.
